# Portfolio Krystal Bellay — version modifiée

## Ce qui a été modifié

- Fond repris du modèle fourni : orbes lumineux, grille légère, ambiance sombre moderne.
- Thème sombre / clair avec bouton dans la navigation.
- Couleur d'accent bleu / rose avec bouton dans la navigation.
- Page d'accueil avec une partie défilante horizontale.
- Page école avec accordéons pour les compétences et une zone de preuves défilante.
- Page communication/design avec catégories nommées, filtres et zone de travaux externes défilante.
- Page CV intégrée directement en HTML, avec bouton de téléchargement PDF.

## Lancer le site dans PyCharm

1. Ouvre le dossier `portfolio_krystal_modifie` dans PyCharm.
2. Clique droit sur `index.html`.
3. Clique sur `Open in Browser`.

## Modifier le CV

Le CV visible est dans `cv.html`.
Le PDF à télécharger est ici :

`assets/docs/cv-krystal-bellay.pdf`

Pour remplacer le PDF, garde le même nom ou modifie le lien dans `cv.html`.

## Ajouter des images

Ajoute tes images dans :

`assets/images/`

Puis remplace les zones :

`IMAGE À REMPLACER`

par exemple :

```html
<img src="assets/images/mon-image.jpg" alt="Description de l'image">
```
